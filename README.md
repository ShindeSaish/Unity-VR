# Unity-VR
Assets for making VR game in unity.
Assests to use -
  1. XR interaction toolkit
  2. Camera movement:
     1. XR origin
     2. XR controller (Action Based)
  3. Player Movement
     1. Locomotion Provider
     2. Continous move provider
     3. Continous turn provider
     4. Character Control driver
 4. Ray
    1. XR ray interactor
    2. XR controller (Action Based)
