# Unity-VR
Assets for making VR game in unity.
Assests to use -
  1. XR interaction toolkit
  2. Camera movement:
     2.1. XR origin
     2.2. XR controller (Action Based)
  3. Player Movement
     3.1. Locomotion Provider
     3.2. Continous move provider
     3.3. Continous turn provider
     3.4. Character Control driver
